# Version Control System - DevVerse

A mini-GitHub implementation built from scratch, demonstrating core version control concepts and functionality.

## Overview

**Version Control System - DevVerse** is an educational version control system project that implements fundamental Git-like features. This system allows users to manage repositories, create branches, track commits, and merge code changes efficiently.

## Features

- **Repository Management**: Initialize and manage version-controlled repositories
- **Branch Management**: Create, switch, and manage multiple branches
- **Commit History**: Track and view complete commit history
- **Merge Operations**: Merge branches and integrate features
- **Authentication**: User login and authentication system
- **Search Functionality**: Search through repository data and commits
- **User Interface**: Console-based UI with intuitive commands
